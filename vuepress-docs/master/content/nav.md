## Active

The defautl active state will bold your active nav item. Nav also has a few classes to allow you to use different styles for the active link. 


### Default

<ul class="nav">
  <li class="nav-item">
    <a class="nav-link active" href="#">Active</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Not Active</a>
  </li>
</ul>

### Pill

With ``.nav-active--pill``

<ul class="nav nav-active--pill">
  <li class="nav-item">
    <a class="nav-link active" href="#">Active</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Not Active</a>
  </li>
</ul>

### Under

With ``.nav-active--under``

<ul class="nav nav-active--under">
  <li class="nav-item">
    <a class="nav-link active" href="#">Active</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Not Active</a>
  </li>
</ul>

## Centered

Add `.flex--center` to center the nav.

<ul class="nav flex--center">
  <li class="nav-item">
    <a class="nav-link active" href="#">Active</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link disabled" href="#" tabindex="-1" aria-disabled="true">Disabled</a>
  </li>
</ul>

## Fill

Add `.nav--fill` to expand the nav items to fill the container.

  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
